# uftc-backend

Capstone projekti

## Ohjeet

1. kloonaa projekti, mene sen kansioon
2. `npm install` asentaa noden moduulit
3. lisää puuttuva .env tiedosto
   - PORT=3001
   - MONGODB_URI=your_mongo_atlas_connection_string
   - SECRET=your_secret_string
